Over-The-Air update handler.

The default password is 123. Beaware that the OTA mechanism works by creating a server on you computer and then instructing the ESP8266 to download the new firmware from that server.
Note: password hashes can be computed using the following commands: echo -n 123 | md5sum

Parameters
msgThe text that will be saved on the backend (panopticon)
user_idThe ID of the user that caused the messaged to be logged
